# Wonderlust

**Wonderlust** is a **real-time travel listings platform** that allows users to explore, view, and book travel stays worldwide. The platform provides a seamless experience for travelers, featuring user authentication, media uploads, and dynamic search functionality.

## Features
- **User Authentication:** Secure sign-up and login using Passport.js.
- **Dynamic Listings:** Add, view, and search for travel stays worldwide.
- **Image Upload:** Upload property images using Cloudinary and Multer.
- **Responsive Design:** Works seamlessly across desktop and mobile devices.
- **Booking System:** Users can book travel stays in real-time.
- **Admin Panel:** Manage listings and users efficiently.
